Welcome

This site hosts the custom report and export bundles developed for Greater Lowell Technical High School (GLTHS), built for the Follett Aspen X2 Student Information System.

Each bundle is a packaged .zip file that can be imported directly into Aspen via Admin → Tools → Bundles → Import Bundle. Reports generate formatted PDF output using JasperReports 6.19.1. Exports generate CSV files for data analysis and spreadsheet use.

Site pages and their descriptions
Page Description
Reports Custom PDF reports deployed as Aspen bundles. Each report uses a JasperReports layout (JRXML), a Java data source, and an input definition for user parameters. Currently includes attendance reports with large student photos.
Exports Custom CSV exports deployed as Aspen bundles. Exports use a Java data source that returns tabular data directly — no JRXML layout needed. Currently includes the Attendance Policy unexcused absence calculator with term filtering and threshold support.
Bundles Download page for all bundle .zip files. Import these into Aspen via Admin → Tools → Bundles → Import Bundle.
